## Spotify Code Quest 2012

The Spotify Code Quest 2012 took place on February 18th, 2012. 

### Problems

  * Toy Railway
  * Evenland
  * Streaming Statistics
  * Greedy Cows
  * Three-State Memory
  * Troll Hunt
  * Smooth Monkey
  * Collapse
